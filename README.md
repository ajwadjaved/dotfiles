map L scrollRight
map h goBack
map l goForward

map x restoreTab

map N restoreTab

map 7 scrollPageUp

map , visitPreviousTab
map . Marks.activateGotoMode

unmap .
unmap /
unmap i
unmap p
unmap w
unmap W
